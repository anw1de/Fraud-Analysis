# 📊 Fraud Analysis (SQL + Python)

## 📌 Описание проекта
Проект направлен на выявление подозрительных пользователей на основе анализа транзакционных данных.

Анализ реализован как последовательно: загрузка данных → подготовка таблицы → аналитический SQL-запрос → выявление аномального поведения.

---

## 🎯 Цель
Найти пользователей с потенциально мошеннической активностью на основе:
- общей суммы покупок  
- частоты заказов  
- среднего чека  
- резких изменений (скачков)  

---

## 📂 Структура проекта

```bash
Fraud-Analysis/
├── data/        # CSV-файл с транзакциями
├── scripts/     # Python-скрипт загрузки данных в БД
├── sql/         # SQL-запрос для анализа
└── README.md
```

💡 **Пояснение:**
- `data/` — исходные данные  
- `scripts/` — подготовка и загрузка  
- `sql/` — вся аналитика  

---

## ⚙️ Логика работы

### 1. Подготовка данных
CSV-файл с транзакциями загружается в базу данных с помощью Python-скрипта.

Скрипт:
- читает CSV  
- создаёт таблицу  
- загружает данные в БД  

---

### 2. Анализ пользователей (SQL)

Основной анализ выполняется SQL-запросом и включает несколько этапов:

**Топ пользователей по сумме покупок**  
Выбираются пользователи с наибольшим оборотом.

**Фильтрация по активности**  
Применяются условия:
- количество заказов в месяц  
- средний чек  

Это позволяет отсеять пользователей с нормальным поведением.

**Поиск скачков**  
Анализируется динамика:
- резкие увеличения суммы покупок  
- аномальные всплески активности  

Такие паттерны могут указывать на мошенничество.

## ▶️ Как запустить

1. Клонировать репозиторий:
```
git clone https://github.com/anw1de/Fraud-Analysis.git
cd Fraud-Analysis
```

2. Установить зависимости:
```
pip install pandas
```

3. Запустить скрипт загрузки:
```
python scripts/load_data.py
```

4. Выполнить SQL-запрос из папки `sql/` в вашей БД
